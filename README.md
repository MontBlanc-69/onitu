Onitu - One cloud to sync them all

Launch it with ```python -m onitu```

Tests and Benchmarks
--------------------

```
$ pip install tox
$ tox -e py27 # tests with python2.7
$ tox -e flake8 # syntaxe
$ tox -e bench # run the benchmarks
```
